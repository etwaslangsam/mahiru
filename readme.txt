○鎮静タイマーについて
	メニューの「鎮静タイマー」は服を脱いだあと（興奮状態）、
	どれくらいの時間で興奮状態が解除されるかの設定です。
	設定された時間になると、興奮状態が解除され服を着ます。
	
	【＋】ボタンを押すごとに、１５秒、３０秒、６０秒と増えていきます
	【−】ボタンを押すごとに、３０秒、１５秒、０秒（興奮解除しない）と減っていきます。
	
○喋り頻度
	お話する頻度を設定します。
	
	【＋】ボタンを押すごとに、６０秒、１８０秒、３００秒と、６００秒と増えていきます
	【−】ボタンを押すごとに、３００秒、１８０秒、６０秒、０秒（喋らない）と減っていきます。
