=============================================

ゴースト「まひるいぢり」

=============================================
製作
=============================================

作者: てにてに
配布: ほっとぺっぱぁ団(https://hotpepar.blogspot.com/)
連絡先: メッセージフォーム(https://forms.gle/LtdNDHAhfSnGBGBt8)

=============================================
概要
=============================================

-可愛い子をお触りするだけのえっちなゴーストです。

=============================================
操作
=============================================

-ダブルクリック
	マウスカーソルが変化する場所:つつき反応

-カーソルを停止させる
	マウスカーソルが変化する場所:見つめ反応

-マウスカーソルが変化する場所でカーソルを左右に動かすとなで反応になります。

-このゴーストは主になで反応がメインです。

-カーソルが変化しても反応自体がない場合があります。

=============================================
謝辞
=============================================

	緋龍華 麒麟　様の【キャラクターなんとか機】を使用させていただいてます。
	蟻の住む木　様の【女の子座り】を使用させていただいてます。
	この場を借りて感謝を申し上げます。
