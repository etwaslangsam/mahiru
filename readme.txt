ゴースト「まひるいぢり」
===============

概要
------------

-可愛い子をお触りするえっちなゴーストです。

-興奮度が５で脱ぐ！

-興奮度が１２以上で絶頂！！

-見つめるだけで絶頂も！！！

操作
------------

-クリック
	マウスカーソルが変化しない場所:ランダムトーク
	マウスカーソルが変化する場所:つつき反応
	
-ダブルクリック
	マウスカーソルが変化しない場所:ゴーストメニューの表示

-カーソルを停止させる
	マウスカーソルが変化する場所:見つめ反応

-マウスカーソルが変化する場所でカーソルを左右に動かすとなで反応になります。

-カーソルが変化しても反応自体がない場合があります。

設定
------------

-トーク
	自分から話す:30〜90秒の間で自動的にランダムトークします。
	話さない:自動でランダムトークしません。
	
謝辞
------------
	緋龍華 麒麟　様の【キャラクターなんとか機】を使用させていただいてます。
	蟻の住む木　様の【女の子座り】を使用させていただいてます。
	この場を借りて感謝を申し上げます。

免責事項
------------

	自己責任でお願いします。
